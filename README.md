# PKGBUILDs

This repository is managed with [`aurpublish`][1].

## Workflow

### 1. Clone

```
git clone git@github.com:thiagowfx/PKGBUILDs.git
aurpublish setup
```

### 2. Pull package

```
aurpublish -p <PACKAGE>
```

### 3. Push package

```
aurpublish <PACKAGE>
```

[1]: https://github.com/eli-schwartz/aurpublish
